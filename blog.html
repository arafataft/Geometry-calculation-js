DOM(Document Object Model) is tree-like representation of HTML in web application. Here HTML element called nodes.

The DOM provides an interface to traverse and modify the nodes. For the dynamic change of web page we need DOM.

It contains methods like getElementById() using this method catch the element by their id and then handle . Other methods are getElementsByClassName() Method. getElementsByName() Method. getElementsByTagName() Method. querySelector() Method.

Event Bubble is a concept in the DOM . When dom method receive an event that event bubbles up to it's parents and can propagated to the inner elements. And event property returns true if an event is a bubbling event.

A callback function is a function passed into another function as an argument. A callback is a function which is to be executed after another function has finished execution. Callbacks make sure that a function is not going to run before a task is completed but will run right after the task has completed. A callback function primary purpose is to execute code in response to an event. These events might be user-initiated, such as mouse clicks or typing.

For event handling i will use callback function. When i want program to react to specific events, such as user input or network requests, I can use a callback function to define what actions should be taken when those events occur.
